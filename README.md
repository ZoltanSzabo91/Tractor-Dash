# Tractor Dash

Arcade-style browser game built with HTML, CSS and Vanilla JavaScript.
Control a tractor, collect pancakes and avoid pedestrians and scooters. The game features multiple difficulty levels and an Endless Mode with increasing difficulty.
The project was created as a learning exercise to practice JavaScript game logic, collision detection, DOM manipulation and state management without using external libraries or frameworks.

---

## Features

- Keyboard controls (Arrow Keys)
- On-screen controls for mobile devices
- Collision detection
- Score tracking
- Multiple difficulty levels
- Endless Mode
- Dynamic speed progression
- Win and lose states
- Debug logging system for collision testing

## Debugging and Testing

To improve collision detection accuracy, I implemented a dedicated debug log panel in Endless Mode.

When a collision occurs, the game displays the positions and movement values of:

- the tractor
- the pedestrian
- the scooter

This allowed me to inspect collision boundaries and calibrate hit detection behavior during development.

---

## How To Play

1. Select a difficulty level.
2. Press Start.
3. Move the tractor using:
   - Arrow keys
   - On-screen buttons
4. Collect pancakes to increase your score.
5. Avoid pedestrians and scooters.
6. Reach the target score or survive as long as possible in Endless Mode.
